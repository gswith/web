--- layout: post title: SSH to remote server without password in CentOS 8 permalink: /ssh-to-remote-server-without-password-in-centos-8/ ---
1. Prerequisites.

$sudo dnf install openssh-client
2. Generate ssh key.

$ssh-keygen

Enter key file: /home/raju/.ssh/my_key
Identification in /home/raju/.ssh/my_key
Public key in /home/raju/.ssh/my_key.pub
3. Copy ssh key to remote server.

$ssh-copy-id -i /home/raju/.ssh/my_key raju@radhas.in
4. Verify.

$ssh -i /home/raju/.ssh/my_key raju@radhas.in hostname

radhas.in
5. The public key will be stored in the file /home/raju/.ssh/authorized_keys at the remote server.

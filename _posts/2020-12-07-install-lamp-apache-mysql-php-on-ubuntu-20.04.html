--- layout: post title: Install LAMP (Apache, MySQL, PHP) on Ubuntu 20.04 permalink: /install-lamp-apache-mysql-php-on-ubuntu-20.04/ ---
1. Install.

$sudo apt update
$sudo apt install apache2 mysql-server php \
          libapache2-mod-php php-mysql w3m
2. Service.

$sudo systemctl restart apache2
3. Verify.

$w3m 127.0.0.1
